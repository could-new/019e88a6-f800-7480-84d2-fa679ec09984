# Free Fire Tournament App

একটি পূর্ণাঙ্গ টুর্নামেন্ট অ্যাপ্লিকেশন, যেখানে ফ্রি ফায়ার প্লেয়াররা ম্যাচ খেলতে এবং টাকা আয় করতে পারবে।

## Features (বৈশিষ্ট্যসমূহ)
- **লগইন ও সাইনআপ (Login & Signup):** নতুন প্লেয়ার বা অ্যাডমিন রেজিস্ট্রেশন।
- **হোম ড্যাশবোর্ড (Home Dashboard):** টুর্নামেন্ট লিস্ট এবং বিস্তারিত তথ্য।
- **ম্যাচে জয়েন (Join Match):** নির্দিষ্ট ফি দিয়ে টুর্নামেন্টে যোগদান।
- **ডিপোজিট ও উইথড্র (Deposit & Withdraw):** ওয়ালেট ব্যালেন্স যোগ এবং টাকা উত্তোলনের সুবিধা।
- **ট্রানজেকশন হিস্ট্রি (Transaction History):** লেনদেনের বিবরণ।
- **প্রোফাইল (Profile):** ব্যবহারকারীর তথ্য।
- **অ্যাডমিন প্যানেল (Admin Panel):** ম্যাচ তৈরি, রুম আইডি/পাসওয়ার্ড সেট করা এবং ইউজার ম্যানেজমেন্ট।

## Tech Stack
- **Frontend:** Flutter (iOS, Android, Web)
- **Backend:** Node.js, Express, MongoDB
- **State Management & Local Storage:** `shared_preferences`, `http`

## Setup Instructions
1. `flutter pub get` চালিয়ে ডিপেন্ডেন্সি ইনস্টল করুন।
2. `lib/main.dart` -এ API URL (backend) সঠিকভাবে বসান।
3. `flutter run` দিয়ে অ্যাপটি চালু করুন।
